## Prompt Template (For Other Destinations)

```

[key visual details specific to the location],
[lighting description], [color tone] tones,
9:16 vertical format, [camera_movement],
shallow depth of field, professional travel videography, no text, no people's faces
```

| Variable | Example (Chiang Mai) | What to change |

| `destination` | Chiang Mai | City/region name |
| `architecture_style` | Thai Lanna wooden | Local architecture |
| `landmark` | Wat Chedi Luang, Doi Suthep | Famous landmarks |
| `nature` | Rice paddies, tropical forest | Local landscape |
| `food` | Khao soi, som tam, satay | Regional dishes |
| `market` | Night bazaar, Sunday walking street | Local markets |
| `activity` | Elephant sanctuary, temple tour | Popular activities |
| `color_palette` | Warm golden, amber, earthy green | Match the destination vibe |
